# 🌦️ Weather App

A simple and responsive Weather Application that provides real-time weather information using the OpenWeather API. Users can search for any city and get current weather details instantly.

---

## 🚀 Features

* 🌍 Search weather by city name
* 🌡️ Real-time temperature display
* 💨 Wind speed and humidity details
* 🌥️ Weather condition (Cloudy, Sunny, Rainy, etc.)
* 📱 Responsive design (works on mobile & desktop)

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript *(or React, if you used it)*
* **API:** OpenWeather API
* **Version Control:** Git & GitHub

---

## 🔗 API Used

We use the **OpenWeather API** to fetch real-time weather data.

* API provides:

  * Current weather data
  * Temperature, humidity, wind speed
  * Weather conditions

👉 Get your API key from: [https://openweathermap.org/api](https://openweathermap.org/api)
